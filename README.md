## Overview

FoT8D is the monorepo for the Field of Truth eight-dimensional quantum-enhanced research platform, including:

- **GaiaFTCL** — Sovereign distributed computing substrate for fusion plasma control, autonomous vehicles, biomedical protocols, and municipal governance
- **GAIAOS** — The operational substrate implementation with nine-cell mesh architecture
- **GaiaFusion** — macOS native Metal-rendered fusion plant control application (GAMP 5 / EU Annex 11 / FDA 21 CFR Part 11 validated)

---

## GaiaFusion Mac Cell

The primary deliverable for CERN deployment lives in `GAIAOS/macos/GaiaFusion/`.

**Quick Start:**
```bash
cd GAIAOS/macos/GaiaFusion
cat README.md  # Complete technical documentation
```

**Key Features:**
- 7 operational states with 18 valid state transitions
- Wallet-based authorization (L1/L2/L3 operator roles)
- 500 plasma particles with temperature-driven color gradient
- WASM constitutional bridge (21 CFR Part 11 §11.200 compliant)
- Pure cyan wireframe rendering via Rust Metal FFI
- HStack/VStack layout with state-forced UI modes
