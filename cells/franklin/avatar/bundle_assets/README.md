# Avatar Asset Bundle

Spec authority: `wiki/AVATAR_CELL_SPEC.md` §2 + §8.

This directory is the *source* of the bundle that gets signed by
`tools/sign_bundle/` and shipped to the app. The shipped bundle lives at
`Contents/Resources/avatar_bundle/` inside the app and is constructed at
build time by `scripts/build_bundle.sh`.

```
assets/
├── meshes/                          # USDZ source (input to bake_mesh)
├── voice/
│   ├── styletts2_franklin_v1.coreml.mlmodelc/   # Production voice model
│   ├── f5_tts_franklin_v1.coreml.mlmodelc/      # Dev fallback
│   └── reference_audio/                         # 30s clip for clone validation
├── pose_templates/
│   ├── viseme/        # 11 — generated by gen_pose_templates
│   ├── expression/    # 12 — generated by gen_pose_templates
│   └── posture/       # 6  — generated by gen_pose_templates
├── illuminants/       # 4 SPD files — generated by gen_illuminants
├── spectral_luts/     # Material reflectance LUTs (skin, fur, fabric, lens)
└── textures/          # Albedo / normal / roughness maps
```

## Build flow

```
USDZ (artist)       ──[usd_to_json]──►  intermediate.json
                                         │
intermediate.json   ──[bake_mesh]─────►  meshes/franklin_passy_v1.fblob
                                         │
StyleTTS 2 weights  ──[coreml_export]─►  voice/styletts2_franklin_v1.coreml.mlmodelc
                                         │
gen_pose_templates  ─────────────────►  pose_templates/{viseme,expression,posture}/*.json
gen_illuminants     ─────────────────►  illuminants/*.json
                                         │
                                         ▼
                                    sign_bundle
                                         │
                                         ▼
                            build/avatar_bundle/manifest.yaml + every asset
```

## What ships

The signed `build/avatar_bundle/` directory is what the runtime reads at boot.
Everything in `assets/` here is the *unsigned source* — running
`verify_bundle` against this directory will fail because the manifest hasn't
been signed yet.
