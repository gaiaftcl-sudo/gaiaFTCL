**Quorum rules (mesh) — verbatim from program `README`:**

- CALORIE (value produced) requires ≥ 5 cells
- CURE (deficit healed) requires ≥ 5 cells
- REFUSED (gate closed) may be declared by any single cell
- A cell that cannot reach quorum enters REFUSED autonomously

**C-003 (substrate) one-liner (verbatim from program `README`):** When S⁴ and C⁴ disagree, **the substrate wins. Always.**

**Narrative (Fusion wiki):** the mesh is a **Klein bottle** invariant: the graph of cell-to-cell connections has **no boundary**. There is no entry point and no exit point from the perspective of a vQbit packet. A measurement emitted from any cell propagates through the mesh and returns to its origin **transformed by the quorum** — without a privileged gateway node.

**No boundary ⇒ consequence:** if quorum is lost, the manifold does not “open” a side exit; transitions fail closed toward **REFUSED** / non-commitment, staying inside the same topological obligation.
